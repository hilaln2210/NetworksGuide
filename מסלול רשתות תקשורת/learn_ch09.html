ויזואליזציה
מה קורה כשמחשב א' שולח ל-ב'?
💡
Hub — למה כמעט לא קיים היום?
Hub = ל-100 מחשבים ברשת → 100 × 99 = 9,900 זוגות שיכולים לשלוח ולהתנגש. Switch פתר את הבעיה — כל פורט = domain נפרד.
🧠
Switch חכם — לומד מבלי שיגידו לו
Switch חדש = Flooding (כמו Hub). אחרי שכל ישות שולחת מסגרת — Switch יודע היכן היא. ללא הגדרה ידנית. MAC → Port נבנה אוטומטית.
🌍
Router — לא רואה MAC, לא מעביר ARP
ARP = Layer 2 בלבד. Router לא מעביר ARP בין רשתות. לכן ל-Default Gateway שולחים ARP, ולא לשרת המרוחק.
🏗️
Switch מודרני — גם Router?
Managed Switches ו-Layer 3 Switches יכולים לבצע routing. הביתי router שלכם = Switch + Router + WAP + Firewall יחד.
