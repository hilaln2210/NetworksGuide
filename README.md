# מדריך רשתות - למידה אינטראקטיבית

© 2025 Hila · [MIT License](LICENSE)

מדריך למידה אינטראקטיבי לרשתות מחשבים, כולל:
- **דפדוף בעמודים** - ניווט קדימה ואחורה
- **סיכומי פרקים** - בסוף כל פרק
- **שאלות הבנה** - עם תשובות והסברים (המדריך מדגים)
- **הדגמות** - הסברים מעשיים שהמדריך מציג

## הרצה

```bash
npm install
npm run dev
```

ואז פתחי בדפדפן: http://localhost:5173

## מקור התוכן

התוכן מבוסס על הספר **רשתות מחשבים** (מטה הסייבר הצה"לי, גרסה 2.2):
- [networks.pdf](https://www.lamed-oti.com/school/oe/networks/networks.pdf)

המדריך כולל 6 פרקים ראשונים מהספר, בעיבוד פורמט למידה אינטראקטיבית – הסברים, הדגמות, סיכומים ושאלות הבנה (המדריך מדגים את התשובות).
