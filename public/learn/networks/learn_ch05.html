מה זה Scapy ולמה צריך אותו?

בפרק 2 למדנו Sockets — לתקשורת ברמת שכבת האפליקציה. Scapy מאפשרת לעבוד ברמות נמוכות יותר: לבנות פקטות IP, TCP, Ethernet, לשלוח אותן, ולהסניף ולנתח פקטות ברשת.

🔍 הסנפה

לתפוס פקטות מהרשת ולעבד אותן בקוד Python

🛠 בניית פקטות

לבנות פקטות מדויקות עם שליטה על כל שדה

📤 שליחה

לשלוח פקטות ולקבל תגובות מהרשת

🧪 ניתוח

לקרוא ולשנות שדות בכל שכבה של פקטה

⚠ שימו לב

Scapy מיובאת בקובץ py כך: from scapy.all import *

🔍 הסנפה עם sniff()

הפונקציה sniff() היא הכלי המרכזי להסנפה. היא blocking — לא תחזור עד שיתפסו מספיק פקטות.

# הסנפה פשוטה — 2 פקטות
packets = sniff(count=2)
packets.summary()

# הסנפה עם filter — רק DNS
def filter_dns(packet):
    return DNS in packet and packet[DNS].opcode == 0

# הסנפה עם prn — פעולה על כל פקטה
def print_query(pkt):
    print pkt[DNSQR].qname

sniff(count=10, lfilter=filter_dns, prn=print_query)

🔎 גישה לשכבות ושדות בפקטה

# בדיקה האם שכבה קיימת
if DNS in packet:
    ...

# גישה לשכבה ספציפית
packet[DNS]        # שכבת DNS
packet[IP]         # שכבת IP
packet[TCP]        # שכבת TCP
packet[Raw]        # מידע גולמי

# גישה לשדה ספציפי
packet[DNS].opcode     # 0 = Query, 1 = Response
packet[DNSQR].qname    # שם הדומיין
packet[DNSQR].qtype    # 1 = A record
packet[IP].src         # כתובת מקור
packet[IP].dst         # כתובת יעד
packet[IP].ttl         # TTL

# הצגה מלאה
packet.show()          # פירוט כל שכבה
packet.summary()       # שורה קצרה

ℹ show() לעומת הצגה ישירה

כשכותבים packet[DNS] מקבלים ייצוג. show() מסביר את המשמעות (כמו Wireshark). לגישה לערך: packet[DNS].opcode מחזיר 0 (לא "QUERY").

🛠 בניית פקטות

Scapy ממלא שדות ריקים אוטומטית בערכים הגיוניים (TTL=64, version=4, checksum וכו'). אנחנו מגדירים רק מה שרוצים לשנות.

# פקטה IP בסיסית
my_pkt = IP()
my_pkt.show()         # Scapy ממלא הכל

# פקטה עם שדות מוגדרים
my_pkt = IP(dst='10.1.1.1', ttl=5)

# Resolving — Scapy מבצע DNS אוטומטי
my_pkt = IP(dst='www.google.com')

# גישה לשדה
my_pkt.src    # '127.0.0.1'
my_pkt.dst    # 'www.google.com'

# שינוי שדה
my_pkt.dst = '8.8.8.8'

💡 Scapy מציג רק שדות ששונו

כשמדפיסים my_pkt (ללא show), Scapy מציג רק שדות שונים מברירת מחדל. זה עוזר לראות בדיוק מה שינינו.

🧱 שכבות — אופרטור /

מחברים שכבות עם /. זה מייצג Encapsulation — שכבה מעל שכבה.

# IP + TCP
my_pkt = IP() / TCP()

# Ethernet + IP + TCP
my_pkt = Ether() / IP() / TCP()

# Ethernet + IP + TCP + HTTP (Raw)
my_pkt = Ether() / IP(dst='1.2.3.4') / TCP(dport=80) / Raw("GET / HTTP/1.0\r\n\r\n")

# IP + TCP עם פרמטרים
my_pkt = Ether() / IP(dst='1.2.3.4', ttl=10) / TCP(dport=80)

Layer 2 — Ether()

שכבת הקו. כתובות MAC.

Layer 3 — IP()

שכבת הרשת. כתובות IP, TTL.

Layer 4 — TCP() / UDP()

שכבת תעבורה. פורטים.

Raw()

תוכן גולמי (Payload). HTTP וכו'.

📤 שליחת פקטות

# שליחה ב-Layer 3 (OS מוסיף Layer 2)
send(my_pkt)          # שלח, לא מחכה לתגובה

# שליחה + קבלת תגובה ב-Layer 3
answered, unanswered = sr(my_pkt)
ans = sr1(my_pkt)      # רק תגובה ראשונה

# שליחה ב-Layer 2 (שליטה מלאה)
sendp(my_pkt)         # כולל Ethernet header

📋 תרגיל 5.1 — הסנפת DNS

המטרה: הדפסת כל שאילתות DNS (סוג A) בזמן גלישה.

from scapy.all import *

def filter_dns(packet):
    return (DNS in packet and
            packet[DNS].opcode == 0 and    # Query
            packet[DNSQR].qtype == 1)         # A record

def print_query_name(pkt):
    print pkt[DNSQR].qname

sniff(count=10, lfilter=filter_dns, prn=print_query_name)

💡 שרשרת הפעולות

1. ipconfig /flushdns — לנקות מטמון DNS
2. הרץ sniff
3. גלוש בדפדפן
4. תראה כל שאילתת DNS מודפסת אוטומטית

📋 תרגיל 5.2 — הסנפת HTTP GET

Scapy לא מכיר HTTP — מתייחס אליו כ-Raw. מסננים לפי תחילת המחרוזת.

def http_get_filter(packet):
    return (TCP in packet and
            Raw in packet and
            str(packet[Raw]).startswith('GET'))

def print_url(pkt):
    data = str(pkt[Raw])
    # מחלץ URL מתוך בקשת GET
    print data.split('\r\n')[0]

sniff(count=20, lfilter=http_get_filter, prn=print_url)

⚠ למה לא HTTP in packet?

Scapy אינו מכיר HTTP כשכבה רשמית — כל תוכן HTTP מופיע כ-Raw. לכן מסננים לפי TCP + Raw + startswith('GET').
