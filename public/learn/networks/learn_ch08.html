🎯 תפקיד שכבת הקו (Layer 2)

שכבת הקו אחראית על העברת מידע בין שתי ישויות המחוברות באופן ישיר — hop אחד בלבד. לא מטפלת בנתיב המלא, רק בצעד הנוכחי.

💡 ההבדל מול Layer 3

Layer 3 (IP): כתובות מקור ויעד = נקודות הקצה הסופיות (מחשבי ↔ Ynet).
Layer 2 (Ethernet): כתובות = hop נוכחי בלבד (מחשבי ↔ נתב, ואז נתב ↔ נתב הבא).

המימוש נמצא בכרטיס הרשת: כרטיס Ethernet ↔ Ethernet, כרטיס WiFi ↔ WiFi. כרטיסים מסוגים שונים לא מתקשרים ישירות זה עם זה.

❓ למה Preamble ו-CRC32 לא ב-Wireshark?

Preamble מטופל ע"י כרטיס הרשת לפני שהמסגרת מגיעה לתוכנה. CRC32 נבדק ואז מוסר ע"י כרטיס הרשת — אם תקין, עובר הלאה ללא ה-CRC.

💡 עיקרון מרכזי

בשכבה 2: src/dst = hop נוכחי. בשכבה 3: src/dst = נקודות הקצה הסופיות. בנתב הבא: src MAC = כרטיס הנתב, dst MAC = הנתב הבא אחריו — אבל ה-IP נשאר זהה!

🧭 ARP — Address Resolution Protocol

ARP ממפה כתובת IP → כתובת MAC. נחוץ כי כדי לבנות Ethernet frame צריך MAC יעד, לא רק IP.

מחשב A →
ARP Request — "מי יש את MAC של 10.0.0.2? תגיד ל-10.0.0.1"
נשלח ל-Broadcast (ff:ff:ff:ff:ff:ff). כולם מקבלים.
מחשב B →
ARP Reply — "אני! ה-MAC שלי הוא AA:BB:CC:DD:EE:FF"
נשלח Unicast ישירות למחשב A (לא Broadcast).

🧠 ARP Cache — מטמון

מערכת ההפעלה שומרת מיפויי IP→MAC במטמון כדי לא לשאול שוב ושוב.
arp -a = הצגת ה-cache.
arp -d <IP> = מחיקת רשומה (דורש הרשאות Admin).

❓ כשיעד מחוץ ל-Subnet — למי שולחים ARP?

אם IP היעד אינו ב-Subnet — שולחים ARP ל-Default Gateway (הנתב), לא ליעד הסופי. הנתב מטפל מכאן.

📡 Hub לעומת Switch

Hub — שכבה 1 (פיזית)

• לא מכיר MAC, לא מבין Ethernet

• מעביר כל מסגרת לכל הפורטים (חוץ ממקור)

• בעיות: פרטיות, עומס, התנגשויות

• כרטיס הרשת של כל מחשב בודק בעצמו אם המסגרת מיועדת אליו

Switch — שכבה 2 (קו)

• מכיר MAC, יכול לחשב Checksum

• שולח כל מסגרת רק לפורט הנכון

• לומד בזמן ריצה: בונה טבלת MAC→Port

• Switch חדש = לא יודע → מתנהג כ-Hub (Flooding)

כיצד Switch לומד — טבלת MAC

📋 תהליך הלמידה

  1. A שולח ל-B (Switch לא מכיר B) → שולח לכל הפורטים (Flooding)
  2. Switch לומד: כתובת MAC של A → Port 1
  3. B שולח ל-A → Switch מכיר A → שולח רק לPort 1
  4. Switch לומד: MAC של B → Port 8
  5. מעתה: כל מסגרת ל-A → רק Port 1, ל-B → רק Port 8

💻 Scapy — שכבה 2

send() לעומת sendp()

send() = Layer 3. Scapy בונה Ethernet לבד.

sendp() = Layer 2. אנחנו שולטים ב-Ethernet.

# Layer 3 — Scapy ממלא MAC אוטומטי
send(IP(dst='google.com')/ICMP())

# Layer 2 — שליטה מלאה
pkt = Ether() / IP(dst='google.com') / ICMP()
sendp(pkt)

סינון לפי MAC

MY_MAC = 'd4:be:d9:d6:0c:2a'

def filter_mac(frame):
    return (Ether in frame and
            frame[Ether].dst == MY_MAC)

sniff(count=10,
      lfilter=filter_mac,
      prn=lambda f: print(f[Ether].src))
