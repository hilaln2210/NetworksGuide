04 — client ב-rlist

לקוח ב-rlist = נתונים זמינים

כשלקוח קיים מופיע ב-rlist — יש נתונים לקרוא. recv() מיד תחזיר נתונים (לא תחסום). אם מחזירה "" — הלקוח התנתק → הסר מ-open_client_sockets.

05 — messages_to_send

messages_to_send — שליחה מדורגת

לא ניתן לשלוח לכל לקוח מיד — Socket אולי לא מוכן. פתרון: שמור הודעות ב-messages_to_send, שלח רק ל-Sockets ב-wlist.

messages_to_send
messages_to_send = []  # list of (socket, data)

def send_waiting_messages(wlist):
    for message in messages_to_send[:]:
        (client_socket, data) = message
        if client_socket in wlist:
            client_socket.send(data)
            messages_to_send.remove(message)

08 — Cheat Sheet

Cheat Sheet — select() Server

server_select.py — תבנית מלאה
import socket, select

server = socket.socket()
server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
server.bind(('', 9001))
server.listen(5)
open_clients = []
messages_to_send = []  # [(socket, data), ...]

while True:
    rlist, wlist, _ = select.select(
        [server] + open_clients,
        open_clients, [])

    for sock in rlist:
        if sock is server:
            client, _ = server.accept()
            open_clients.append(client)
        else:
            data = sock.recv(1024)
            if not data:
                open_clients.remove(sock); sock.close()
            else:
                messages_to_send.append((sock, data.upper()))

    for sock, msg in messages_to_send[:]:
        if sock in wlist:
            sock.send(msg)
            messages_to_send.remove((sock, msg))
09 — שאלות בחינה

שאלות בחינה נפוצות

❓ למה server_socket נמצא ב-rlist ולא ב-wlist?
server_socket.accept() היא פעולת קריאה (מחכה ללקוח חדש). wlist = sockets מוכנות לכתיבה — לא רלוונטי לserver socket שרק מאזין.
❓ מה יקרה אם לא נסיר socket ממת מ-open_clients?
select יחזיר אותה כ-readable בכל iteration, recv תחזיר b"", נכנס ללולאה אינסופית. תמיד: if not data: remove + close.
❓ למה לא לשלוח ישירות ב-rlist loop?
Socket אולי לא מוכן לכתיבה (buffer מלא). שליחה ל-socket לא מוכן → blocking או שגיאה. messages_to_send + wlist = בטוח.
