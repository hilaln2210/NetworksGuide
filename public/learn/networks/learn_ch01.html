STEP 01
Request — בקשה

ענן האינטרנט
הבקשה עוברת hop-by-hop דרך עשרות נתבים, ממחשבך עד לשרת בקליפורניה.

Facebook מחזיר HTML+CSS+JS. הדפדפן מרכיב ומציג את העמוד.
👁️
View Page Source — לראות את ה-Response
לחץ על הכפתור הימני של העכבר בכל אתר → "View page source" — תראה את ה-HTML שהשרת שלח. זהו בדיוק הResponse שקיבל הדפדפן.

🔍
גילוי IP עם ping (כתוצר לוואי)
ping נועד לבדיקת קישוריות — לא לגילוי IPs. אבל כתוצר לוואי מציג את ה-IP של היעד לפני שמתחיל לפנג'.

DNS — ספר הטלפונים

Domain Name System

בני אדם זוכרים שמות (facebook.com). מחשבים צריכים כתובות IP. DNS = שירות תרגום בזמן אמת.

1
דפדפן שואל שרת DNS
nslookup www.facebook.com → DNS Query
2
DNS מחזיר כתובת IP
www.facebook.com → 31.13.72.65
3
דפדפן שולח Request ל-IP
GET / HTTP/1.1 → 31.13.72.65:80
4
שרת Facebook מחזיר Response
HTTP 200 OK + HTML + CSS + JS

05 —

GeoIP — מיקום גיאוגרפי

IP → מיקום פיזי

מאגרי נתונים ממפים כתובת IP לעיר/מדינה. לא רשמיים, לא מדויקים — אבל שימושיים לסקירה כללית.

⚠️ מקרה אמיתי — GeoIP יצר אסון
חוות חקלאית בקנזס, ארה"ב, קיבלה מאות פשיטות משטרה שגויות. הסיבה: ספק GeoIP מיפה את כל כתובות ה-IP הלא-ידועות לנקודת ברירת מחדל — שנפלה בדיוק על הבית של האיכר. GeoIP ≠ מיקום אמיתי מדויק. לעולם לא.

מקור: fusion.net/story/287592/internet-mapping-glitch-kansas-farm
06 —

כלי שורת הפקודה

Command Line Tools
🏓
ping
קישוריות + זמן תגובה
שולח "בקשת הד" ומחכה לתשובה. בודק אם יעד מחובר ומציג זמן הלוך-חזור. כתוצר לוואי — מציג את כתובת ה-IP.
C:\> ping www.facebook.com
# מציג IP + זמן תגובה ב-ms
🗺️
tracert
מסלול מלא עד ליעד
מציג את כל ה-Hops בדרך — כל נתב שהחבילה עברה דרכו. הדגל -d מבטל תרגום DNS.
C:\> tracert -d www.google.com
📋
nslookup
תרגום שם דומיין → IP
שאילתת DNS ישירה. מחזיר כתובת IP (או כמה) השייכות לשם דומיין נתון. כלי DNS טהור.
C:\> nslookup www.facebook.com
🌍
Visual Traceroute
מסלול על מפת עולם
GeoIP + traceroute = רואים את המסלול של החבילה על מפה. כל Hop = נקודה על הגלובוס.
# yougetsignal.com
# tools/visual-tracert

סיכום מושגים

Glossary
מושגהגדרהדוגמה
Client תוכנה שמבקשת שירות — יוזמת Request הדפדפן שלך
Server תוכנה שמספקת שירות — עונה ב-Response שרת Facebook
Request בקשה שהלקוח שולח לשרת "תן לי את דף הבית"
Response תגובה שהשרת שולח ללקוח HTML + CSS + JS
IP Address כתובת מזהה ייחודית — 4 bytes, כל byte 0-255 31.13.72.65
DNS מערכת תרגום שם דומיין ← → כתובת IP facebook.com → 31.13.72.65
Hop מעבר חבילה בין שני רכיבים מחוברים ישירות מחשב → נתב ביתי = hop 1
GeoIP מיפוי IP → מיקום גיאוגרפי (לא מדויק) geoiptool.com
WWW World Wide Web — אוסף עמודי אינטרנט מקושרים כל האתרים שגולשים אליהם
