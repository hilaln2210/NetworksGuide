⚡ שלב 1 — יצירת Hash וניסיון שבירה
# צור MD5 Hash לסיסמה "password123" echo -n "password123" | md5sum # → 482c811da5d5b4bc6d497ffa98491e38 # שמור ב-file echo "482c811da5d5b4bc6d497ffa98491e38" > hash_md5.txt # שבור עם RockYou wordlist (Dictionary Attack) hashcat -m 0 -a 0 hash_md5.txt /usr/share/wordlists/rockyou.txt # הוסף Rules (ל-1337speak, numbers suffix וכד') hashcat -m 0 -a 0 hash_md5.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule # עכשיו BCRYPT — אותה סיסמה, כמה זמן? # צור BCRYPT online: bcrypt-generator.com hashcat -m 3200 -a 0 hash_bcrypt.txt rockyou.txt

✅ תשובות — Password Cracking
01
MD5 "password123": פחות מ-1 שנייה. כלול ב-RockYou wordlist. GPU RTX 4090 שובר 164 מיליארד MD5 hash לשנייה.
02
"P@ssw0rd!": עם Rules — עדיין דקות בודדות. Leet-speak substitutions הם הראשונות שHashcat מנסה. לא מגן.
03
"correct-horse-battery-staple": 4 מילים רנדומליות ≈ 10⁵² אפשרויות. לא ב-Dictionary, Brute Force לוקח מיליוני שנים. זו הסיסמה החזקה.
04
BCRYPT vs MD5: BCRYPT מכוון להיות איטי. GPU שובר 184 BCRYPT לשנייה (לא מיליארדים). 164 מיליארד / 184 = פי 891 מיליון איטי יותר.
💡 Password Entropy Chart:
• 8 תווים, lowercase: 26⁸ ≈ 208 מיליארד — נשבר בשנייה (MD5)
• 4 מילים רנדומליות: 7776⁴ ≈ 3.6 טריליון — לא ניתן לשבור
• BCRYPT + Salt + 4 מילים = בלתי שביר מבחינה מעשית
⚠️ לעולם אל תשתמש ב: MD5, SHA1, SHA256 לסיסמאות. תמיד: bcrypt, Argon2id, או scrypt עם work factor גבוה.
