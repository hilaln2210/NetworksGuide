✅ תשובות — Nmap
01
Host Discovery: -sn (Ping Scan) שולח ICMP Echo + TCP SYN ל-443 + TCP ACK ל-80. מחזיר רשימת hosts חיים ללא סריקת פורטים. מהיר למיפוי ראשוני של רשת.
02
SYN Scan ("Stealth"): שולח SYN, אם מקבל SYN-ACK → פורט פתוח. שולח RST מיידית (לא משלים Handshake). פחות רעש ב-Logs מ-Full Connect Scan. דורש Sudo.
03
-sV: מנסה לזהות גרסת Service. -sC: מריץ NSE Scripts בסיסיים (Default Scripts). ביחד: nmap -sV -sC -p 22,80,443,3306 <target> = הקומבו הנפוץ ביותר.
06
Metasploitable2 שירותים פגיעים נפוצים: FTP (21) עם anonymous login, SSH (22) עם גרסה ישנה, Samba (139/445) עם ms08-067, MySQL (3306) ללא סיסמה, DVWA (80).
💡 חשוב: סרוק רק רשתות ומערכות שיש לך הרשאה לסרוק. סריקת רשתות ללא אישור היא עבירה פלילית ברוב המדינות. לתרגול: השתמש ב-Metasploitable, TryHackMe, או HackTheBox.
