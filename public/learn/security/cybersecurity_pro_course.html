🔎 מעבדה 1 — Network Scanning עם Nmap EASY
🎯 מטרה: ללמוד לסרוק רשת, לזהות פורטים פתוחים ושירותים פעילים. כלי חיוני לכל Pentest ו-Network Admin.
⚡ תרגיל: סריקת יעד
השתמש ב-Nmap לסרוק את Scanme.nmap.org (יעד לגיטימי לתרגול)
# סריקה בסיסית — גילוי פורטים פתוחים nmap scanme.nmap.org # סריקה עם זיהוי שירותים וגרסאות nmap -sV -sC scanme.nmap.org # סריקת OS Fingerprinting nmap -O scanme.nmap.org # סריקה מהירה של 1000 פורטים נפוצים nmap -T4 -F scanme.nmap.org
⚡ אתגר: פרש את הפלט
מה זה STATE: filtered לעומת open לעומת closed? איזה שירות מסוכן אם הוא פתוח לאינטרנט?
🎣 מעבדה 2 — Phishing Email Analysis EASY
🎯 מטרה: ניתוח Header של מייל חשוד — זיהוי מקור, SPF/DKIM/DMARC, ואינדיקטורים של זיוף.
⚡ תרגיל: ניתוח Header
פתח מייל חשוד (בסביבת sandbox), הצג את ה-Headers המלאים וחפש:
01 בדוק את שדה "Reply-To" — האם שונה מ-"From"?
02 חפש SPF fail / DKIM fail ב-Authentication-Results
03 עקוב אחרי Received: headers מלמטה למעלה — מאיפה באמת נשלח?
04 בדוק לינקים עם URLScan.io לפני לחיצה
05 העלה קבצים מצורפים ל-VirusTotal
⚡ כלים לשימוש
MXToolbox Email Header Analyzer → mxtoolbox.com/EmailHeaders.aspx URLScan.io → בדיקת לינקים חשודים VirusTotal → בדיקת קבצים ו-URLs PhishTool → ניתוח מקיף של פישינג
