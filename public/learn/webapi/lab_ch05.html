RateLimit Headers:
X-RateLimit-Limit: 100 — המגבלה הכוללת
X-RateLimit-Remaining: 73 — כמה נשאר
X-RateLimit-Reset: 1720000060 — מתי מתאפס (Unix timestamp)
Retry-After: 30 — כמה שניות לחכות (בעת 429)
03
ETag Flow: Server חישב MD5 של Response Body → שלח כ-ETag Header. Client שמר. Request הבא: If-None-Match: "abc123". אם לא השתנה → Server מחזיר 304 ללא Body. חסכנו 100% מה-Bandwidth.
💡 ETag vs max-age: max-age = "אל תשאל ל-N שניות". ETag = "שאל, אבל אם לא השתנה קבל 304". stale-while-revalidate = Best of Both Worlds — הגש מה-Cache, ועדכן ברקע.
