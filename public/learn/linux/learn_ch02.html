מה לומדים
pwd, cd — איפה אני ואיך מתנייד
ls, ls -l, ls -la — הצגת קבצים עם פרטים
mkdir, rmdir — יצירה ומחיקה של תיקיות
touch, rm, rm -rf — יצירה ומחיקה של קבצים
cp, mv — העתקה והעברה
. ו-.. ו-~ — מה כל קיצור אומר
סכנה קלאסית
💥 MISTAKE — למתחילים
rm -rf / — הפקודה שמוחקת הכל
אחד הטעויות הקלאסיות: הקלדת rm -rf / במקום rm -rf ./folder. הסלש הראשון = Root = כל המחשב. Linux המודרני חוסם זאת עם --no-preserve-root, אבל הסיפור נשאר אגדה. כלל הזהב: לפני rm -rf, תמיד echo קודם.
ניווט בסיסי
# ← רואים את הנקודות? . = כאן, .. = אחורה
$ cd Documents
$ cd .. # אחורה תיקייה אחת
$ cd ~ # תמיד חזרה ל-Home

# צור מבנה תיקיות שלם בבת אחת
$ mkdir -p projects/web/src
(אין פלט = הצלחה בלינוקס!)

# ls עם פרטים — רואים הכל
$ ls -lh
drwxr-xr-x 3 student student 4.0K Jan 15 projects
